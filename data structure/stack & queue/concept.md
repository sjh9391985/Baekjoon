### 스택과 큐는 배열에서 발전된 형태의 자료구조

# 1. 스택
     - LIFO 자료구조
     - 삽입과 삭제가 한 쪽에서 일어나는 특징
     - push: top 위치에 데이터를 삽입
     - pop: top 위치에 현재 있는 데이터 삭제 후 확인하는 연산
     - peek: top 위치에 현재 있는 데이터 단순 확인 연산
     - 깊이 우선 탐색(DFS), 백트랙킹 종류의 코딩 테스트에 효과적이기에 알아야 함.


# 2. 큐
    - FIFO 자료구조
    - 먼저 들어온 데이터가 먼저 나감.
    - 삽입과 삭제가 양방향에서 이뤄짐.
    - rear: 큐에서 가장 끝 데이터를 가리키는 영역
